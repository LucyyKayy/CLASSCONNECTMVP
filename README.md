ClassConnect

ClassConnect is a multilingual classroom integration platform designed to help migrant and refugee students understand lessons in real time and progressively learn the host country language. The platform provides live translation, auto-generated exercises, and progress tracking for students, teachers, and parents.

Features

Live Translation: Voice and text translation between the host country language and the studentâ€™s home language.

Learning Bridge: Automatically generates exercises and homework to reinforce language learning.

Progress Tracking: Teachers and parents can monitor comprehension and improvement.

Offline Mode: Download lessons and transcripts for offline use.

Multi-language Support: Supports 15 languages including English, German, French, Spanish, and more.
